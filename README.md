# LJT-Homepage Draft

This repository contains a draft academic profile site for Ravi Narayan.

## Reviewing the draft

1. Open `site_profile.yml` to review structured profile data.
2. Read `pages/home.md` for the homepage draft.
3. Check `pages/publications.md` for the publications list.
4. Note that private details (personal phone number, passport number, family contact name) were intentionally excluded and are not published.

## Files

- `site_profile.yml` — structured profile data
- `pages/home.md` — homepage draft
- `pages/publications.md` — publications list
- `.nojekyll` — disables Jekyll processing
