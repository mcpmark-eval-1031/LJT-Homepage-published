We introduce **SynLogic**, a framework for synthesizing high-quality, verifiable reasoning data at scale. SynLogic generates logical reasoning problems with ground-truth solutions that can be automatically verified, enabling large language models to learn complex logical reasoning patterns.

The key contributions of this work include:

* A novel data synthesis framework that produces millions of verifiable logical reasoning examples
* Demonstration that models trained on SynLogic data achieve state-of-the-art performance on logical reasoning benchmarks
* Open-source code and data release
